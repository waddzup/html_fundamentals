Exercise 2.2 Question 3

Does an attribute called src exist? What is the purpose of this attribute and what are the elements it can be applied to?

The src attribute applies to the following html elements:

  • <audio>
  • <embed>
  • <iframe>
  • <img>
  • <input>
  • <script>
  • <source>
  • <track>
  • <video>

The value of the <src> attribute gives the url of some embedded content: audio, video or image..
